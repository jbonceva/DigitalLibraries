# Digital Libraries Repository

This repository contains homework assignments and projects for the Digital Libraries course at FINKI.  
Each project is developed using Greenstone3 and represents a digital library collection with different types of documents, metadata, and search/browse functionality.

---

## Contents

- Collections created using the Greenstone Librarian Interface (GLI)
- Documents in multiple formats (PDF, DOCX, HTML)
- Multimedia content (MP3 audio files, images, embedded videos)
- Metadata for organization and search functionality

## How to Use

1. Download the `.zip` file of the desired collection from this repository or Google Drive
2. Extract it into your local `Greenstone3/collect/` directory
3. Open Greenstone Librarian Interface (GLI)
4. Select "Build Collection"
5. The collection will then be available through the Greenstone web interface
