# Digital Libraries Repository

This repository contains my homework assignments and projects for the Digital Libraries course.
Each project is built using Greenstone3 and represents a digital library collection with different types of documents, metadata, and search/browse functionality.

## Contents

* Collections created with the Greenstone Librarian Interface (GLI)
* Documents in multiple formats (PDF, DOCX, HTML)
* Multimedia content (MP3, images, embedded videos)
* Metadata for organizing and searching

## How to Use

1. Download the .zip file of the desired collection from this repository or Google Drive.
2. Extract it into your local Greenstone3/collect/ directory.
3. Open GLI and select "Build Collection".
4. The collection will then be available through the Greenstone web interface.
