# DigitalLibraries – Leonardo da Vinci Collection

This repository contains my homework project for the Digital Libraries course.  
The project was created using **Greenstone Digital Library Software** and focuses on the life and works of **Leonardo da Vinci**.

## Project Description
The collection includes five HTML documents that present different aspects of Leonardo da Vinci’s biography, inventions, and artistic achievements.  
Metadata has been added to each document to enable search and retrieval inside Greenstone.

## Structure
- `docs/` – HTML documents with content about Leonardo da Vinci  
- `metadata/` – metadata files describing the documents  
- `etc/` – configuration files generated by Greenstone  

## Local Usage
To use this collection in Greenstone, place the folder in: Greenstone3\web\sites\localsite\collect\leonardo_da_vinci_collection
